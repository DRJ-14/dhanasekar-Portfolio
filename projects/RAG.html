Context-Aware Email Assistant

  • Tech Stack: Python, LangChain, ChromaDB, GPT4All Embeddings, Llama.cpp, Hugging Face Transformers, TRL, PEFT/LoRA, bitsandbytes, Jupyter, BeautifulSoup
  • Github URL: Project Link

Built a Retrieval-Augmented Generation (RAG) pipeline to answer natural-language questions over Gmail archives by exporting emails as .mbox, parsing + cleaning HTML bodies (BeautifulSoup), chunking with recursive splitters, generating embeddings (GPT4All embeddings), and indexing in Chroma for semantic retrieval.

Implemented retrieval + generation with LangChain (retriever → prompt → LLM) using local inference backends (GPT4All / Llama.cpp) to ground responses in retrieved email snippets and reduce hallucinations.

Fine-tuned Gemma with QLoRA using Transformers + TRL SFTTrainer + PEFT (LoRA) + bitsandbytes 4-bit, then quantized for efficient local inference and integrated the tuned model into the RAG workflow.

Witreless Transport Protocol

  • Tech Stack: C, Socket Programming, UDP/TCP, Transport Protocols, Linux/macOS, Git/GitHub
  • Github URL: Project Link

Implemented a custom wireless transport protocol featuring sequence numbering, ACK/NAK handling, adaptive retransmission (ARQ), and timeout/RTT estimation to ensure reliable end-to-end delivery under packet loss and variable link conditions.

Built a protocol test harness and evaluation pipeline to generate controlled traffic and collect metrics (throughput, goodput, end-to-end latency, packet loss, retransmission rate), enabling comparative analysis across multiple network scenarios.

Optimized protocol behavior via parameter tuning (window/flow control limits, retransmission timers, congestion/backoff strategy) and validated improvements using repeatable experiments and empirical performance reports.

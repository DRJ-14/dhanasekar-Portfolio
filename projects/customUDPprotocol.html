Custom UDP protocol data Transfer

  • Tech Stack: C, Socket Programming, UDP, Client–Server Architecture, Custom Protocol design, Linux/macOS, Git/GitHub
  • Github URL: Project Link

Engineered a custom application-layer protocol over UDP with packet framing (SOP/EOP), sequence/ID fields, payload length encoding, and checksum validation to ensure message integrity across an unreliable datagram channel.

Implemented reliability mechanisms on top of UDP, including ACK-based delivery semantics, timeout-driven retransmissions (ARQ), and duplicate/out-of-order handling to achieve consistent end-to-end client→server transfer.

Built a metrics-driven test harness to benchmark throughput/goodput, end-to-end latency, packet-loss impact, and retransmission overhead; tuned protocol parameters (timeouts/windowing where applicable) based on empirical results.
